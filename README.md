# 高性能跨语言模板引擎 Crox

## 综述

Crox是一个由JavaScript语言实现的 `高性能` `跨语言` 模板引擎。

Crox模板可以直接在JavaScript环境中使用，也可以翻译成PHP、JSP等其他编程语言的可执行方法，或翻译成Velocity、Smarty等其他模板引擎的源模板。

Crox将保证翻译后的结果具备最佳执行效率。
